# 3. 防抖 节流
多次点击，最后一次才执行
```
function debounce（fn,time）{
let timer=null
 return function(){    
 if(timer) clearTimeout(timer)
timer=setTimeout(()=>{
fn.apply(this,arguments)
},time)
}}
```

}}
